
# Вебсистема підтримки колективного прийняття рішень

Демонстраційний прототип бакалаврського проєкту.
Функціональність:
- створення сесій експертного оцінювання;
- додавання альтернатив та експертів;
- збереження оцінок у SQLite;
- порівняння 5 методів: зважений середній бал, Борда, Кондорсе, Делфі, парні порівняння;
- розрахунок Kendall W, коефіцієнта варіації та ентропії;
- генерація PDF-звіту.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Після запуску відкрийте у браузері: http://127.0.0.1:5000/
