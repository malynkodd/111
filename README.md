# Вебсистема підтримки колективного прийняття рішень

Веб-застосунок для організації та проведення сесій експертного оцінювання альтернатив. Підтримує п'ять методів агрегування думок, метрики узгодженості, Делфі-процедуру з міжраундовим зворотним зв'язком та генерацію PDF-звітів.

## Запуск

1. Клонувати репозиторій:
   ```bash
   git clone <url>
   cd 111
   ```

2. Встановити залежності:
   ```bash
   pip install -r requirements.txt
   ```

3. Створити `.env` з секретним ключем (скопіювати `.env.example`):
   ```bash
   cp .env.example .env
   # відредагувати SECRET_KEY у .env
   ```

4. Запустити додаток:
   ```bash
   python app.py
   ```
   Додаток буде доступний за адресою `http://127.0.0.1:5000`.

## Методи оцінювання

1. **Зважений середній бал** — зважена сума балів з урахуванням компетентності кожного експерта.
2. **Метод Борда** — кожен експерт ранжує альтернативи; бали підсумовуються (n−1 за 1 місце, 0 за останнє).
3. **Метод Кондорсе** — підрахунок попарних перемог з ваговими коефіцієнтами.
4. **Метод Делфі** — ітеративний метод з міжраундовим зворотним зв'язком; консенсус через `0.7·поточний + 0.3·попередній`.
5. **Метод парних порівнянь (AHP-подібний)** — побудова матриці попарних порівнянь та вектора пріоритетів.

## Метрики узгодженості

- **Kendall W** — коефіцієнт конкордації (0–1, де ≥0.7 вважається достатньою узгодженістю).
- **Коефіцієнт варіації (CV)** — відносний розкид оцінок по кожній альтернативі.
- **Ентропія** — міра неоднорідності розподілу оцінок.
- **Kendall τ / Spearman ρ** — кореляція між ранжуваннями різних методів.
- **Консенсусне ранжування** — агрегація результатів усіх методів через медіану рангів.
- **Виявлення аутлайерів** — eксперти, чиє середнє відхилення від групового перевищує поріг (за умовчанням 2σ).
