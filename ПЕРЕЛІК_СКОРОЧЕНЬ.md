## Математичні позначення

| Позначення | Визначення |
|-----------|-----------|
| `a` | Альтернатива |
| `B(a)` | Сума балів Борда для альтернативи `a` |
| `CV(a)` | Коефіцієнт варіації оцінок для альтернативи `a` |
| `D(a)` | Агрегована оцінка методу Делфі для альтернативи `a` |
| `H(a)` | Ентропія оцінок для альтернативи `a` |
| `m` | Кількість експертів у сесії |
| `M` | Матриця попарних порівнянь (метод АНР) |
| `n` | Кількість альтернатив у сесії |
| `p_j` | Нормалізована оцінка j-го експерта (`p_j = s_j / Σs_k`) |
| `r_i(a)` | Ранг альтернативи `a` у списку i-го експерта |
| `R(a)` | Сума рангів альтернативи `a` по всіх експертах |
| `R̄` | Середня сума рангів усіх альтернатив |
| `S` | Сума квадратів відхилень сум рангів від середнього значення |
| `S(a)` | Зважений середній бал альтернативи `a` |
| `s_i(a)` | Оцінка i-го експерта для альтернативи `a` |
| `s_i_cur(a)` | Оцінка i-го експерта в поточному раунді для альтернативи `a` |
| `s_i_prev(a)` | Оцінка i-го експерта в попередньому раунді для альтернативи `a` |
| `W` | Коефіцієнт конкордації Кендалла |
| `w_i` | Коефіцієнт компетентності (вага) i-го експерта |
| `wins[a]` | Кількість перемог альтернативи `a` у методі Кондорсе |
| `σ(a)` | Генеральне стандартне відхилення оцінок для альтернативи `a` |
| `μ(a)` | Середнє арифметичне оцінок для альтернативи `a` |
| `Σ` | Знак підсумовування |
| `log₂` | Логарифм за основою 2 |

## Позначення методів та алгоритмів

| Позначення | Метод |
|-----------|-------|
| Метод 1 | Зважений середній бал (Weighted Score Method) |
| Метод 2 | Метод Борда (Borda Count Method) |
| Метод 3 | Метод Кондорсе (Condorcet Method) |
| Метод 4 | Метод Делфі (Delphi Method) |
| Метод 5 | Метод парних порівнянь / АНР (AHP-like Method) |
